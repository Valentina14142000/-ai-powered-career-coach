# **App Name**: CareerPilot AI

## Core Features:

- AI Resume Semantic Analysis: Extracts complex professional experience and technical skills from uploaded resumes using AI for optimized job alignment.
- Intelligent Opportunity Matcher: An AI tool that reasons through candidate experience and job descriptions to provide high-fidelity score rankings for open roles.
- Tailored Cover Letter Architect: An AI tool that synthesizes specific bullet points from your resume with the target job posting to write unique, persuasive cover letters.
- Interactive Mock Interview Coach: Generative AI persona that conducts simulated interviews and provides constructive feedback on response sentiment and relevance.
- Visual Application Kanban: A centralized dashboard to drag-and-drop job applications through customized pipeline stages with real-time analytics.
- Cross-Platform Authentication: Secure sign-on via Google and Microsoft accounts integrated with the database to maintain cross-device persistence.
- Subscription & Resource Tiers: Tiered access controls for monthly plans, managed via a cloud-based SQL database architecture.

## Style Guidelines:

- Primary color: Electric Indigo (#6E6EFF), chosen to represent innovation and career progression while providing high visibility on dark backgrounds.
- Background color: Deep Shadow Navy (#14141F), a dark, sophisticated hue that provides a modern, high-tech workspace feel.
- Accent color: Deep Azure Blue (#478FFF), providing a contrasting analogous tone for actionable UI elements and progress bars.
- Headline font: 'Space Grotesk', a geometric sans-serif that evokes a technical and forward-thinking personality. Body font: 'Inter', selected for its exceptional legibility in dense data dashboards.
- Minimalist line icons with subtle gradients to denote AI actions, keeping the interface uncluttered and professional.
- A modular card-based dashboard layout utilizing CSS Grid to accommodate varied analytic widgets and the primary application board.
- Smooth micro-interactions on hover and subtle fade-in sequences for AI-generated text components to enhance the conversational feel.
