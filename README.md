# CareerPilot AI 🚀
### Your AI-Powered Career Architect & Job Search Copilot

CareerPilot AI is a modern, high-performance web platform designed to help job seekers navigate the complex landscape of modern hiring. Built with **Next.js 15**, **Tailwind CSS**, and **Google Gemini AI (via Genkit)**, it provides professional-grade tools to optimize every stage of the career transition.

## ✨ Key Features

- **🧠 Semantic Resume Analysis**: Our AI doesn't just look for keywords; it understands the semantic context of your experience, extracting complex technical skills and professional achievements.
- **🎯 Intelligent Opportunity Matcher**: Get a precision match score for any job description. The AI provides detailed reasoning on why a role is a good fit and identifies potential gaps.
- **✍️ Tailored Cover Letter Architect**: Generate high-conversion, personalized cover letters that align your specific career highlights with the requirements of a target role.
- **🎙️ Interactive Mock Interview Coach**: Practice your responses with an AI coach that provides instant feedback on sentiment, relevance, and actionable suggestions for improvement.
- **📊 Application Pipeline Tracker**: A visual Kanban-style board to manage your job search pipeline, from application to offer.

## 🛠 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **AI Engine**: [Google Genkit](https://firebase.google.com/docs/genkit) + [Gemini 2.5 Flash](https://deepmind.google/technologies/gemini/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn/UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Firebase App Hosting](https://firebase.google.com/docs/app-hosting)

## 🚀 Getting Started

### Prerequisites

- Node.js 20 or later
- A Google Cloud/Firebase Project for AI capabilities

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Valentina14142000/-ai-powered-career-coach.git
   cd -ai-powered-career-coach
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables:
   Create a `.env` file and add your Google AI API Key:
   ```env
   GOOGLE_GENAI_API_KEY=your_api_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the results.
